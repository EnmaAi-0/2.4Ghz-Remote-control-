# 2.4Ghz-Remote-control-
这是一个基于MM32SPIN27芯片制作的SI24R1遥控设备，可以与逐飞科技的无线串口模块直接通讯。可以在不改变原PCB的情况下远程控制智能车，包括方向，参数等（需要二次开发）。
代码部分只包含基础的通讯，AD包含全套原理图，PCB。
